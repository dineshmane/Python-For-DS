# trustpilot
Scraping and analysis in python for trustpilot.com reviews.

# Dependencies

 - Python 3.6
 - requests 2.12.4
 - BeautifulSoup
 - Pandas
 - Matplotlib
 - WordCloud
 - NLTK
 - Numpy
 - PIL

# Usage
 1) Set up up and configure ipynb to scrape the page you want the reviews and save into CSV file
 2) Use the saved csv file for further analysis
